# Ghostnote Labs — Legal

Public-facing legal documents for [Ghostnote Labs LLC](https://ghostnote-labs.github.io/legal/) products.

## What's here

- [`privacy.md`](privacy.md) — Privacy policy. Covers health data access, on-device storage, Anthropic (Claude) as an AI subprocessor, user rights. Published to `/privacy`.
- [`terms.md`](terms.md) — Terms of service. Covers license grant, conduct, disclaimer, governing law. Published to `/terms`.
- [`index.md`](index.md) — Landing page at `/`.

## Publishing

Rendered by GitHub Pages with the default Cayman Jekyll theme. Any push to `main` updates the live site within ~1 minute.

## Updating

Edit the canonical sources in the private `eSmelser/numen` repo at `docs/legal/`. When ready to publish updated policies, copy here, prepend the Jekyll frontmatter block, commit + push.

The markdown files here are the PUBLISHED versions. The ones in `numen/docs/legal/` are drafts that track version history alongside the code.

## Canonical URLs

- https://ghostnote-labs.github.io/legal/
- https://ghostnote-labs.github.io/legal/privacy
- https://ghostnote-labs.github.io/legal/terms

These URLs are what gets registered with Apple App Store Connect and Google Play Console as the app's legal URLs.
